# Kart-over-Videreg-ende-Skoler-i-Vestland
Kart over Videregående Skoler i Vestland er en interaktiv webapp som viser alle VGS-skoler i Vestland fylke på et kart. Brukere kan filtrere etter region og eierskap, søke etter skoler. Hver markør viser detaljert info: adresse, programmer, nettside. Bygget med moderne webteknologier for responsiv opplevelse.
